# @sequential/file-operations

Safe file operations with atomic writes, path validation, and protection against traversal attacks.

## Installation

```bash
npm install @sequential/file-operations
```

## Usage

```javascript
import {{ main }} from '@sequential/file-operations';

const result = main(/* args */);
```

// Basic usage
