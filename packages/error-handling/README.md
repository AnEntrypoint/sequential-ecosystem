# @sequential/error-handling

Centralized error handling with AppError class, error factory functions, and structured logging utilities.

## Installation

```bash
npm install @sequential/error-handling
```

## Usage

```javascript
import {{ main }} from '@sequential/error-handling';

const result = main(/* args */);
```
