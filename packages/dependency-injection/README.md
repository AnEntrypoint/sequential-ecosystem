# @sequential/dependency-injection

Lightweight DI container for managing service dependencies with singleton pattern and circular dependency detection.

## Installation

```bash
npm install @sequential/dependency-injection
```

## Usage

```javascript
import {{ main }} from '@sequential/dependency-injection';

const result = main(/* args */);
```

## API

### Main Export

Main function/class provided by this package.

## Examples

```javascript
// Basic usage
const value = main(input);

// With options
const result = main(input, {{ /* options */ }});
```
